#Lista de Tarefas com Firebase
##Projeto Realizado para a cadeira de Programação de Dispositivos Móveis

# Funcionalidades Implementadas

### **Sistema de Autenticação**
- Login com email e password
- Registo de uma nova conta
- Logout
- Verificação automática da sessão ao abrir a app
- Mensagens de erro em português

### **Gestão de Tarefas**
- Listar tarefas do utilizador atual
- Associar tarefa a uma determinada categoria
- Adicionar nova tarefa
- Marcar/desmarcar como concluída
- Tarefas riscadas quando concluídas
- Cada utilizador vê apenas as suas tarefas
